Introduction
============

DUnitRunner extends and uses the native python unittest framework. DUnitRunner is a specialized unit test
runner that allows passing command line arguments to the test case itself rather than only passing arguments to
the unit test runner. Output generated by DUnitRunner includes JUnit compliant XML, standard unittest text, and
ascii text report for each evaluation statement.


Example Construction Syntax
===========================

Using DUnitRunner via the command line:

.. code-block:: python

  srun python vDUnitUnit.py evalStdOut=True evalStdOutOnlyFail=True


Using DUnitRunner via the python test script:

.. code-block:: python

  if __name__ == '__main__':
    DUnitTestRunner.DUnitRunner(evalStdOut=True, evalStdOutOnlyFail=True)
